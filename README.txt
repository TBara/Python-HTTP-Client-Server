Before running this program, ensure all required packages are installed 
by running command in your terminal:
    
    pip3 install -r requirements.txt

To run the client file, execute command:

    python3 http_client.py

To run the server file, run command:

    python3 http_server.py

then navigate to http://127.0.0.1:11234/ or http://localhost:11234/
in your web browser.
